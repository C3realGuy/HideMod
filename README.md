HideMod v1.0 (alpha)

Hey here is my first plugin for wedge :)
Its a simple HideMod. It supports [hide] (you need to like) and [hide-reply].

Installation:

Drop "hide" folder in your /wedgefolder/plugins and activate it in the adminpanel.
